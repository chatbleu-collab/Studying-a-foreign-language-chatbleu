# 🗣️ AI Language Coach

학습자의 수준에 맞춰 **절대 어렵게 말하지 않는** AI 회화 선생님 PWA.
영어(CEFR A0~B2+)와 일본어(JLPT N5~N2+)를 지원합니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| 배치 테스트 | 첫 실행 시 레벨 테스트로 시작 레벨 자동 추정 |
| 적응형 AI 튜터 | 현재 레벨에 맞는 어휘·문장 길이로만 대화 (Gemini API) |
| 오프라인 튜터 | API 키 없이도 커리큘럼 스크립트 기반 가이드 대화 가능 |
| 12개 레슨 모드 | 자유회화·가이드레슨·롤플레이·발음·단어(SRS)·문법·듣기·말하기챌린지·JLPT·TOEIC·여행·일일미션 |
| SRS 단어장 | 간격 반복(SM-2 간이형)으로 복습 시점 자동 계산 |
| 진도 추적 | XP·스트릭·레벨 승급·일/주/월 리포트·약점 분석 |
| 음성 | Web Speech API — TTS(듣기)·STT(말하기), 미지원 브라우저는 텍스트 폴백 |
| PWA | 홈 화면 추가·오프라인 동작·다크모드·모바일 퍼스트 |

## 빠른 시작

### 1) 바로 사용 (빌드 결과물)
`dist/` 폴더 전체를 HTTPS 정적 호스팅(GitHub Pages, Netlify 등)에 업로드하면 끝.
하위 경로(예: `username.github.io/repo/`) 배포도 지원합니다(상대 경로 빌드).

### 2) 개발 환경
```bash
npm install
npm run dev      # 개발 서버
npm run build    # dist/ 생성 (tsc 타입체크 포함)
```

## Gemini AI 연결 (선택)

1. 브라우저에서 `https://aistudio.google.com/apikey` 접속 → Google 계정으로 무료 API 키 발급
2. 앱의 **설정 → AI 튜터 연결**에 키 입력
3. 무료 티어는 Flash 계열 모델만 해당(기본값 `gemini-2.0-flash`). 모델명이 바뀌면 설정에서 직접 수정 가능

> 키는 **본인 기기의 LocalStorage에만** 저장됩니다. 키가 없거나 호출이 실패하면 자동으로 오프라인 튜터로 전환됩니다.

## 브라우저 지원

- **권장: Chrome(Android/PC)** — STT·TTS 모두 동작
- Safari(iOS): TTS 동작, STT 제한적 → 텍스트 입력 자동 폴백
- 음성 인식은 마이크 권한 허용 필요

## 알려진 한계 (정직 고지)

- 오프라인 튜터는 **커리큘럼 스크립트 기반**이라 자유 주제를 이해하지 못합니다(화면에 고지됨).
- 상위 레벨(B1/B2, N3/N2) 커리큘럼 스크립트는 하위 레벨보다 단원 수가 적습니다(확장 예정).
- 발음 채점은 STT 인식 결과와의 **단어 일치율** 기반 근사치이며 음소 단위 평가가 아닙니다.
- Gemini 모델명·무료 정책은 Google 사정으로 변경될 수 있어 설정에서 수정 가능하게 열어두었습니다.
